Destrinchador de IPs versão 0.5

- Consulte a ajuda do programa para entender os formatos de entrada aceitos e a saída gerada.
- Por se tratar de um applet Java, este programa exige que esteja instalada e dividamente configurada, no computador onde for executado, a máquina virtual Java -JVM.
- Durante o desenvolvimento do programa foi utilizado o Java 2 Standard Edition 1.5 -J2SE 1.5-, portanto o autor recomenda que ele esteja instalado no computador, caso o usuário deseje recompilar o código-fonte.
- Para se executar o programa, tendo a JVM ou o J2SE -que inclui a JVM- instalados, basta abrir o arquivo Applet.htm que acompanha este programa em um navegador de Internet. Há um aplicativo de visualização de applets incluido no J2SE chamado appletviewer. Pode-se visualisar o programa com ele, através do comando:
$ appletviewer Applet.htm
É indispensável dizer que, caso se deseje ver o aplicativo através do navegador de Internet, o mesmo deve estar devidamente configurado para acessar a JVM e que no caso de se utilizar o appletviewer, o J2SE deve estar configurado no sistema.
